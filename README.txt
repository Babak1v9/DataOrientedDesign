Damit das Programm auf die richtigen Files zugreift muss man zuerst das ArrayCreator-Projekt ausführen, damit die Datenquellen für die Sortieralgorithmen erstellt werden. 
Wir haben uns für diese Lösung entschieden, weil wir somit die Größe des Projektes klein halten, aber trotzdem zum Vergleich immer auf die selben Files zum Sortieren zugreifen.
